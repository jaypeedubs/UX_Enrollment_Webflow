Action buttons
Pinned to the detail header — always visible regardless of which tab is active. Each opens a confirmation dialog before firing. Actions adjust per status: an accepted applicant only shows Reject.
Tabs
Application — personal info + locked-field indicators + program Q&A answers.
CV — signed URL iframe, download link.
Admin Notes — free-text, auto-saves, timestamp + author.
Timeline — application_events audit log.
Locked fields
Fields with locked_fields set show a lock icon and muted text. Reminds reviewers which data is immutable — first name, last name, and program lock on first submit.
Notes privacy
Amber warning banner at top of Admin Notes tab. Stores to applications.admin_notes — a column the applicant's RLS policy cannot read. Last-saved timestamp shows who edited.

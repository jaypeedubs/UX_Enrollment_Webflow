Section 5 preview — Auth flow (no visual needed)

Admin login uses Supabase auth (email + password or magic link). On sign-in, the app checks the user's JWT for app_metadata.role === "admin". If the claim is missing, the user is shown an "Access denied" screen and signed out. Setting the role is a one-time Supabase dashboard operation per admin user.
